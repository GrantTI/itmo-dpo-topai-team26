# Модуль 1. Backend

---

## Аннотация

Модуль посвящён освоению практических навыков проектирования и разработки серверной части веб-приложений для последующей интеграции ML-пайплайнов. В рамках занятий студенты изучают архитектуру клиент-серверных систем, учатся визуализировать распределённые приложения в нотации C4 (уровни 1 и 2) с корректным определением компонентов (прокси, бэкенд, БД, фронтенд) и сетевых взаимодействий.

Основная практическая часть модуля (занятия 1.2–1.6) направлена на создание REST API с использованием фреймворка FastAPI. Студенты осваивают трехуровневую организацию кода (маршруты → контроллеры → сервисы), интегрируют API с ML-пайплайнами, автоматически документируют эндпоинты через OpenAPI и внедряют единый механизм обработки ошибок.
