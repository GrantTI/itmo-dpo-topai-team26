# Модуль 3. Интеграция и развёртывание

Директория содержит описания контрольно-измерительных материалов (КИМ) модуля 3 дисциплины «Технологии разработки и развёртывания моделей машинного обучения в клиент-серверных приложениях».

## Назначение модуля

Модуль направлен на формирование умений интегрировать клиентскую и серверную части приложения, подключать сервисы инференса моделей машинного обучения к клиент-серверной архитектуре и развёртывать приложение в контуре, приближённом к production, с использованием контейнеризации и обратного прокси.

**Объём:** 26 а.ч. (лекции — 4; лабораторные работы — 14; самостоятельная работа — 8).

## Компетенции и уровни

| Код | Название компетенции | Индикатор | Уровень |
|---|---|---|---|
| PL-1 | Способен применять язык программирования Python для решения задач в области ИИ | PL-1.1 | С |
| LC-5 | Способен применять и (или) проектировать различные инструменты и инженерные практики промышленной разработки, развёртывания, эксплуатации и мониторинга систем ИИ | LC-5.1 | Б |

**Дескрипторы в контексте модуля.** По завершении модуля обучающийся:

- согласовывает контракт REST API между клиентом и сервером, организует HTTP-взаимодействие и настраивает политику CORS и reverse-proxy;
- реализует на FastAPI обращение к внешнему или mock-сервису инференса с управлением секретами через переменные окружения и обработкой ошибок и таймаутов;
- собирает и запускает клиент-серверное ML-приложение средствами Docker и Docker Compose, выполняет конфигурацию через переменные окружения и публикацию через Nginx, проверяет работоспособность развёрнутого контура.

## Место в структуре дисциплины

Модуль опирается на результаты модулей 1 (Backend, REST API, применение моделей через API) и 2 (Frontend, интерфейсы взаимодействия с API) и обеспечивает подготовку развёрнутого контура для тестирования и защиты проекта в модуле 4.

Последующее развитие компетенции LC-5 до среднего уровня относится к дисциплинам блока жизненного цикла систем ИИ (MLOps / DevOps), включающим оркестрацию контейнеров, непрерывную интеграцию и развёртывание, а также практики сопровождения моделей в продуктивной среде.
