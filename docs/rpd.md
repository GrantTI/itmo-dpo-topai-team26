# Рабочая программа дисциплины

## 1. Общие сведения

| Поле | Значение |
|---|---|
| Название дисциплины | Технологии развертывания моделей машинного обучения в клиент-серверных приложениях |
| Образовательная программа | Искусственный интеллект и машинное обучение  |
| Уровень образования / формат ДПО | Бакалавриат |
| Объем | 108 ч. / 3 з.е. |
| Формат обучения | Очный |
| Форма промежуточной аттестации | Зачет |

## 2. Цель и задачи

**Цель:** Формирование у студентов комплексных знаний и практических навыков разработки веб-приложений, интегрирующих технологии искусственного интеллекта и машинного обучения.

**Задачи:**

1. Изучение основ веб-технологий и принципов клиент-серверного взаимодействия
2. Приобретение навыков эффективного проектирования архитектуры
3. Развитие навыков работы с механизмами обработки и визуализации данных
4. Формирование умения создавать интерфейсы для взаимодействия с ML-моделями в реальных индустриальных условиях


## 3. Пререквизиты

- Объектно-ориентированное программирование (PL-2, PL-3, средний)
- Обработка данных на Python (PL-1, базовый)
- Машинное обучение (ML-3, ML-4, ML-7, MF-4, BD-1, LC-5, базовый)
- MLOps&DevOps (LC-4, средний)
- Технологии тестирования программного обеспечения (PL-3, средний)
- Компьютерные сети (LC-5, базовый)
