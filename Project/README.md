# Проектная работа

Проектная работа — интегральный КИМ, позволяющий проверить несколько компетенций и индикаторов в условиях комплексной практической задачи.

### 1. Общие сведения
**Раздел:** Проект  
**Форма контроля:** защита индивидуального проекта  
**Максимальный балл:** 100  
**Назначение:** оценка результатов выполнения проектного задания, качества программной реализации, корректности применения инструментов разработки и уровня владения материалом при защите.
**Контрольно-измерительные материалы (КИМ):** индивидуальное проектное задание и его защита. Оценивание проводится по 100-балльной шкале по критериям, отражающим постановку задачи, архитектуру решения, реализацию backend на Python, работу с данными, развертывание модели машинного обучения, применение инструментов промышленной разработки, качество кода и уровень защиты проекта.

### 3. Объект оценивания
Оценивается индивидуальный проект в виде клиент-серверного приложения, обеспечивающего развертывание и использование модели машинного обучения, работу с данными, обработку запросов, документирование и реализация работы с API, обоснование выбранного решения и защиту решения.

### 4. Содержание проекта
**Проект должен включать:**
- постановку задачи, решаемой при помощи средств машинного обучения (рекомендуемый список моделей прилагается);
- поиск данных (датасет) для выбранной задачи;
- описание архитектуры клиент-серверного приложения;
- реализацию backend-сервиса на Python;
- поиск, выбор и интеграция работы приложения с данными (датасетом);
- API для загрузки и обработки данных;
- обоснование выбора модели машинного обучения (логистическая регрессия, дерево решений, случайный лес, градиентный бустинг и т.д.), применительно к задаче;
- API для применения модели машинного обучения;
- получение метрик качества и/или параметров модели;
- обработку ошибок и исключений;
- документацию API;
- подготовку и защиту результатов проекта.


**Рекомендуемый список моделей машинного обучения:**
- логистическая регрессия;
- дерево решений;
- метод опорных векторов (SVM);
- наивный Байесовский классификатор;
- случайный лес;
- градиентный бустинг;
- полносвязная нейронная сеть (FCNN)*;
- свёрточная нейронная сеть (CNN)*;

\* - в качестве задания повышенной сложности
