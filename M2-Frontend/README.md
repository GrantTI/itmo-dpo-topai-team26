# Модуль 2. Frontend

Директория содержит MD-файлы с описаниями КИМ для модуля 3. В этом файле можно сформировать таблицу со списком КИМ для модуля, их краткими аннотациями и ссылками на соответствующие MD-файлы.

В данном модуле представлены следующие виды КИМов по дисциплине "Технологии развертывания моделей машинного обучения в клиент-серверных приложениях" 
- **Тесты** для закрепления и проверки знаний по теоретическим вопросам. Тесты проводят по окончании лекций и направлены на получение обратной связи от студентов к преподавателю о понимании материала лекции. В рейтинговой системе имеют самый низкий коэффициент полезности. 
- **Лабораторные работы** предполагают работу по инструкции с некоторой исследовательской частью. Лабораторные работы защищаются студентами перед преподавателем в устной форме. При выполнении лабораторной работы студенту не запрещается пользоваться LLM, однако критерием качества выполнения заданий лабораторной работы является понимание своих действий и ответы на вопросы преподавателя. В рейтинговой системе средний коэффициент полезности.
- **Индивидуальное задание** выдаются студентам на самостоятельное выполнение. Задания даются индивидуально для каждого студента и не предполагают идентичных результатов у двух и более студентов. В рейтинговой системе высший коэффициент полезности.
- **Workshop (мастерская)** совместное выполнение командами решения поставленной задачи. Предполагают одинаковые по типу задачи, но выполнение с помощью различных средств.
