# Фонд оценочных средств по дисциплине «Технологии развертывания моделей машинного обучения в клиент-серверных приложениях»

Этот репозиторий предназначен для открытого и воспроизводимого представления фонда оценочных средств (ФОС) по дисциплине «Технологии развертывания моделей машинного обучения в клиент-серверных приложениях». Здесь объединены рабочая программа дисциплины, модель измерения результатов обучения, контрольно-измерительные материалы (КИМ), критерии оценивания, методические рекомендации и используемые образовательные ресурсы. Результаты обучения связываются с компетентностно-ролевой моделью в области искусственного интеллекта (КРМ версии 3.0).

## 1. О дисциплине

**Цель дисциплины.** Формирование у студентов комплексных знаний и практических навыков разработки веб-приложений, интегрирующих решения, разработанные с применением технологии искусственного интеллекта и машинного обучения.

**Основные задачи.**
1. Изучение основ веб-технологий и принципов клиент-серверного взаимодействия
2. Приобретение навыков эффективного проектирования архитектуры
3. Развитие навыков работы с механизмами обработки и визуализации данных
4. Формирование умения создавать интерфейсы для взаимодействия с ML-моделями в реальных индустриальных условиях

**Структура и содержание.** Дисциплина состоит из 4 модулей: "Backend", "Frontend", "Интеграция и развертывание", "Защита проекта". Обучение включает лекции / лабораторные работы / проект / самостоятельную работу и завершается дифференцированным зачетом.

**Пререквизиты и связи с другими дисциплинами.** 
- Объектно-ориентированное программирование (PL-2, PL-3, средний) 
- Обработка данных на Python (PL-1, базовый) 
- Машинное обучение (ML-3, ML-4, ML-7, MF-4, BD-1, LC-5, базовый) 
- MLOps&DevOps (LC-4, средний) 
- Технологии тестирования программного обеспечения (PL-3, средний) Компьютерные сети (LC-5, базовый)

**Планируемые результаты обучения.** По завершении дисциплины обучающийся сможет [ЗАПОЛНИТЬ: 3–6 проверяемых результатов обучения, сформулированных через наблюдаемые действия].

Полное описание приведено в [РПД](docs/rpd.md).
